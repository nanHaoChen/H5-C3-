早年经历

武磊（Wu Lei），1991年11月19日出生于江苏省南京市，祖籍山东济南平阴，中国足球运动员，司职前锋，现效力于西班牙足球甲级联赛的皇家西班牙人足球俱乐部。 [1-3] 2006年，14岁10个月的武磊在中乙联赛登场，成为中国职业足球赛场史上最年轻的球员。2012年，武磊单赛季打入17个进球，以中甲本土射手王的身份带领上海东亚冲超成功。 [4] 2013年，武磊在中国对澳大利亚的比赛中打入个人在国家队的首球。 [5] 2014年，武磊以12个联赛进球蝉联中超本土最佳射手。2015年11月7日，武磊获颁本土最佳射手称号。 [7] 2016年，武磊获得亚洲足球先生提名。2017年11月11日，当选2017中超联赛最佳本土射手。 [8] 2017年11月，武磊连续第二年入围亚洲足球先生三甲候选之列。 [9] 2018年3月18日，武磊在中超联赛对阵广州富力的比赛中上演“大四喜”。 [10] 2018年3月，《世界足球》评选世界500强球员，武磊成榜上仅有的中国人。 [11] 2018年11月7日，武磊随上海上港队夺得中超联赛冠军；11月21日，武磊包揽2018赛季中超金球奖和金靴奖，并入选中超最佳阵容。 [12-13] 2019年1月28日，武磊加盟西甲皇家西班牙人足球俱乐部。2019年3月2日，武磊收获西甲生涯首球。 [14] 2019年5月，武磊的进球送西班牙人进入欧联杯资格赛。 [15] 2020年1月5日，武磊成为历史上首位攻破巴萨球门的中国球员。 [16] 2020年1月17日，武磊蝉联中国金球奖。 [17] 2019年8月20日，福布斯中国发布2019名人榜，武磊位列第55。 [18-19] 2020年6月13日，武磊攻破阿拉维斯球门，成为五大联赛首位感染新冠康复后进球的球员。

晚年经历

2006年9月2日，在中乙联赛第13轮比赛中，年仅14岁零10个月的前锋武磊代表上海东亚队在上海东亚3比5不敌丽江东巴的比赛中替补出场，创造中国足球职业联赛史上最年轻出场球员纪录 [25] 。 2008年8月30日，在中甲联赛第15轮比赛中，武磊代表上海东亚队与青岛海利丰一役中攻入一球，当时年仅16岁零289天。他也成为了继曹赟定之后，中国职业足球联赛第二年轻的进球者 [25] 。 2008年的中甲联赛，武磊打进了4个球，是上海东亚队内第二号射手 [28] 。 2009年，在第十一届全国运动会中，武磊代表上海队打入4个球，为上海队夺得男子足球甲组冠军立下了汗马功劳 [28] 。 2009年中甲联赛，武磊打进6球，是上海东亚队的队内最佳射手 [28] 。 2012年，武磊在中甲联赛中攻入17个进球，帮助上海特莱士队冲超成功 [29] 。 2013年3月8日，在中超首轮上海上港对阵北京国安的比赛中，武磊打入个人在中超的首粒进球 [30] 。 武磊在联赛中 武磊在联赛中(3张) 2013年6月2日，在中超第12轮比赛中，上海上港以6-1大胜上海申鑫的比赛中，武磊上演帽子戏法并送上了2次助攻 [31] 。 2013年8月17日，中超第21轮，上海上港客场对战天津泰达队，武磊再次完成独中三元的帽子戏法。帮助上海上港3:2战胜天津泰达队。 2013年9月27日，中超第26轮，武磊在37分钟内上演帽子戏法，帮助上海上港6-1大胜青岛中能。武磊一个赛季上演了三次帽子戏法，这在中国足球史上是史无前例的，武磊成为中国顶级联赛单赛季完成三次“帽子戏法”这一壮举的第一人 [31] 。 。2016年5月8日，2016赛季中超联赛第8轮的，石家庄永昌主场对阵上海上港。48分钟，蔡慧康中路带球分到左路，左路队员将球传至禁区，埃弗拉顶了一下，将球顶在了对方防守球员的身上，结果皮球反弹后，武磊机敏射门球进。这也是武磊职业生涯正式比赛的第100粒进球 [33] 。 2016年5月21日，武磊与上海上港俱乐部完成续约 [34] 。5月24日，亚冠联赛十六进八第二回合，武磊第91分钟补射绝杀，上海上港1比0战胜东京FC，凭借客场进球优势，进军亚冠八强 [35] 。7月31日，中超联赛第20轮，广州富力3比3上海上港，第19分钟埃弗拉左路长传，武磊在大门后点推射得分，打进了个人在职业联赛（中超和中甲）里的第100球，并连续四个中超赛季实现联赛进球上双。第90分钟，武磊打进点球实现个人帽子戏法 [36] 。 2017年6月21日，中国足协杯16进8的比赛，武磊补时救主，上海上港经过17轮点球大战，以16比15（点球比分15比14）战胜苏州东吴，杀入足协杯八强 [37] 。7月22日，中超联赛第18轮，武磊梅开二度，帮助上港2比2战平恒大 [38] 。 2018年3月3日，中超联赛首轮，上海上港主场以8-0战胜升班马大连一方，夺取双线4连胜。奥斯卡、武磊先后完成帽子戏法 [39]
